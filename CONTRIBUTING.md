requires [yarn](https://yarnpkg.com/en/)

### setup

```sh
git clone https://github.com/kedromelon/baffle-react.git
cd baffle-react
yarn
```

### develop

```sh
yarn start
```

### lint

```sh
yarn lint
```

### test

```sh
yarn test
```

```sh
yarn test:coverage
```

```sh
yarn test:watch
```

### format

```sh
yarn prettier
```

### build

```sh
yarn build
```
